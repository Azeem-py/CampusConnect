Gradient Descent Analysis

Consider the following optimization problem for the cost function $J(\theta)$. We aim to minimize the variance across the batch size $m$.

$$ \theta_j := \theta_j - \alpha \frac{1}{m} \sum_{i=1}^{m} (h_\theta(x^{(i)}) - y^{(i)})x_j^{(i)} $$

Using the above update rule, we converge to the local minima provided the learning rate $\alpha$ is sufficiently small.
